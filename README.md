# La Cuisine de Gagny

Site vitrine de La Cuisine de Gagny — bistrot-traiteur, 153 boulevard Chave, Marseille 5e.

Réalisé par **LeStud**.

## Structure
Site single-file : tout (HTML, CSS, JS, fonts, images, motifs) est embarqué dans `index.html` en base64. Aucune dépendance externe, aucun build.

## Déploiement
- Hébergé sur Netlify (publish directory : `.`, pas de commande de build)
- Domaine : lacuisinedegagny.com

## Stack
Vanilla HTML / CSS / JS · Netlify

Voir `LELAB.md` pour la direction artistique complète et la mémoire projet.
